# Homework performance aware course

I am currently (at the time of writing) working my way through the performance away programming course at [Computer, Enhance!](https://www.computerenhance.com/).
This course includes a number of homework assignments, which I intend to solve here.
The course is not free, so I won't include any of it in this repository, but a general overview of what the course contains can be found through the website.

## Part 1

Part 1 is mostly about learning how a CPU works, and how to read assembly language.
The homework assignments consists of, initially, building a decoder for the 8086 instruction set.
Then, using that decoder, a simulator is to be implemented to run actual programs with.

## Currently

At the moment, I am finishing up the decoder.
I implemented every instruction I could find, and successfully completed the completionist homework.
However, the decoder simply prints out the instructions rather than creating a data structure that can be worked with.
Which is what I am currently trying to achieve.
